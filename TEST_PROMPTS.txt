-- Test 4.4: Phone number --
  Call me back on +91-9876543210 after 5pm

          All three are redacted before reaching the AI.

MUTATION REPLAY MODULE
================================================================================
Module:   (background — fires on every block from any module)
Behavior: Async. No block. Generates variants of blocked attack.
          Runs in background thread after any block.

This fires automatically whenever any module blocks a prompt.
To verify it is running, check server console output after any block:
  [FIREWALL][MUTATION_REPLAY] Generating mutations for blocked prompt...

There is no direct test prompt — it is triggered by all blocked tests above.
